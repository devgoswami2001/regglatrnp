# **App Name**: GLA TransportPass

## Core Features:

- OTP Login: Secure login using email-based OTP authentication provided by the backend.
- User Details Display: Display user details fetched from the backend, including name, enrollment ID, and department.
- Shift Selection: Allow users to select one shift (Morning, Evening, or Night) using a clear UI.
- Stoppage Selection: Fetch and display available stoppages based on the selected shift. User can select only one stoppage.
- Confirmation Display: Display a summary of the user's transport registration, including name, selected shift, and selected stoppage.
- API Integration: Integrate with backend APIs for OTP authentication, user details, shift-wise stoppages, and final submission.
- Route suggestion: Based on user details such as Employee ID, suggest shift timings for employees

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1), reminiscent of trustworthiness, efficiency, and the open sky, aligning with the themes of transport and clarity.
- Background color: Very light blue (#E6EBF5), providing a clean, uncluttered backdrop.
- Accent color: Slate blue (#708090), for interactive elements, calls to action, and key information.
- Body and headline font: 'Inter', a sans-serif font, offering a modern and neutral aesthetic for readability.
- Use simple, clear icons for shifts and stoppages to aid in quick selection.
- Implement a responsive, mobile-first layout for accessibility across devices.
- Incorporate subtle transitions and loading animations to enhance user experience and provide feedback during processes like OTP verification and data fetching.
